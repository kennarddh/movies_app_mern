## Server

```bash
cd server
```

## Available Scripts

In the project directory, you can run:

### Run dev

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Start build

```bash
npm start
```

### Lint

```bash
npm run lint
```

### Prettier fix

```bash
npm run prettier
```

### Prettier check

```bash
npm run prettier-check
```
